# Treinamento de Prisma/Node.JS

Introdução ao Prisma, versão 6, com Node.JS, versão 24.x, com o objetivo de aprender os conceitos e usaos básicos deste ORM.

Sistema desenvolvido em TypeScript.

## Aplicação

Cursos técnicos do Instituto Federal de São Paulo, campus __Caraguatatuba__

### Cursos

* Técnico de Informática Integrado ao Ensino Médio
  * Disciplina: Linguagem de Programação - CARLPRO
* Técnico de Informática para Internet - Concomitante
  * Disciplina: BackEnd 2 - CARBKE2

## Instalação

Após descompactar o projeto rodar:

```bash
npm install
```

Depois copiar o arquivo `.env.example` para `.env`:

```bash
cp .env.example .env
```

E alterar as variáveis de ambiente.

## Uso

Para rodar o projeto, utilize o comando padrão:

```bash
npm run dev
```
